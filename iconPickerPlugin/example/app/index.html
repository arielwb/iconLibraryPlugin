Turistur icon library

Instruções:

Inclua o script 

<script src="pathTo/ttIcons.js"></script>

Adicione um elemento com a classe tt-icon-picker
Neste elemento, adicione o atributo data-controls com o id do input que deve ter seu valor setado por este botão 

<button class="tt-icon-picker" data-controls="icon-category" ></button>

Adicione um input com o id informado no botão 

<input type="hidden" id="icon-category"></input>
